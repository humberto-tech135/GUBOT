# GUBOT 🤖

Protótipo de cão-guia robótico de baixo custo.

## Objetivo

Desenvolver uma solução acessível para auxiliar a locomoção de pessoas com deficiência visual por meio da detecção de obstáculos e navegação autônoma.

## Status

🚧 Em desenvolvimento

## Componentes

- Arduino Uno
- Sensores ultrassônicos HC-SR04
- Ponte H
- Motores DC
- Bateria 18650
